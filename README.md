# Погодная станция

Инфраструктура для мониторинга погодных данных в реальном времени. Проект демонстрирует полный цикл работы с данными: генерация, хранение, визуализация и глубокий анализ.

## Архитектура проекта
Весь стек развернут в **Docker** и включает в себя:
* **PostgreSQL**: Реляционная база данных для хранения истории погоды.
* **Python Generator**: Кастомный сервис, имитирующий сенсоры (температура, влажность, давление).
* **Redash**: BI-инструмент для оперативного мониторинга и создания SQL-дашбордов.
* **Jupyter Notebook**: Среда для исследовательского анализа данных (EDA) на языке Python.

## Как запустить

1. **Клонируйте репозиторий:**
   ```bash
   git clone <ссылка_на_твой_репозиторий>
   cd my_analytics_project

2. **Запустите контейнеры:**
   ```bash	
   docker-compose up -d

3. **Инициализируйте базу данных Redash (только при первом запуске):**
   ```bash
   docker-compose run --rm redash create_db

4. **Остановка проекта:**
   ```bash
   docker-compose stop

## Анализ в Jupyter

В папке notebooks/ находится файл analysis.ipynb. Скрипт выполняет следующие задачи:

* Подключение к БД через SQLAlchemy.
* Визуализация трендов температуры, влажности и давления с помощью Seaborn.
* Расчет базовых статистических показателей.
